Vector magnitude#

The magnitude of a vector \(\overrightarrow{AB}\) is the distance between the two points \(A\) and \(B\) which we can calculate using an extension of Pythagoras’ theorem.

Definition: Vector magnitude

The magnitude of a vector \(\mathbf{a} = (a_1, a_2, \ldots, a_n)\) denoted by \(|\mathbf{a}|\) is calculated using

(8)#\[|\mathbf{a}| = \sqrt{\sum_{i=1}^n a_i^2} = \sqrt{a_1^2 + a_2^2 + \cdots + a_n^2}.\]

Vector magnitude is also known as the Euclidean norm. Note that \(|\mathbf{a}|=0\) if and only if \(\mathbf{a}=(0, 0, \ldots, 0)\).

Example 3.2

Calculate the magnitudes of the following vectors

(a)   \(\mathbf{u} = \begin{pmatrix} 1 \\ 2 \\ 3 \end{pmatrix}\)     (b)   \(\mathbf{v} = \begin{pmatrix} 5 \\ -12 \\ 0 \end{pmatrix}\)     (c)   \(\mathbf{w} = \begin{pmatrix} 1 \\ 0 \\ 1 \end{pmatrix}\)

Solution

(a)   \(|\mathbf{u}| = \sqrt{1^2 + 2^2 + 3^2} = \sqrt{1+4+9} = \sqrt{14}\);

(b)   \(|\mathbf{v}| = \sqrt{5^2 + (-12)^2 + 0^2} = \sqrt{25+144+0} = \sqrt{169} = 13\);

(c)   \(|\mathbf{w}| = \sqrt{1^2 + 0^2 + 1^2} = \sqrt{1+0+1} = \sqrt{2}\).

Unit vectors#

For every non-zero vector \(\mathbf{a}\) there exist a unique unit vector which is a vector in the same direction as \(\mathbf{a}\) and whose magnitude is 1.

Definition: Unit vectors

A unit vector is a vector with a magnitude of 1.

Proposition: Normalising a vector

Any non-zero vector can be scaled to transform it into a unit vector by dividing all its coordinates by its magnitude

(9)#\[\hat{\mathbf{a}} = \frac{\mathbf{a}}{|\mathbf{a}|}.\]

This process is called normalising a vector. Unit vectors are denoted with a caret above the vector name, i.e., \(\hat{\mathbf{a}}\) which is read as ``\(\mathbf{a}\) hat’’.

Proof

Let \(\mathbf{a}\) be a non-zero vector

\[\begin{align*} \left|\frac{1}{|\mathbf{a}|}\mathbf{a}\right| &= \left|\frac{1}{|\mathbf{a}|}\right| |\mathbf{a}| \\ &= \frac{1}{|\mathbf{a}|} |\mathbf{a}| \qquad \text{(since $|\mathbf{a}|>0$)}\\ &= 1. \end{align*}\]
□

Example 3.3

Find the unit vector parallel to the following:

(a)   \(\mathbf{u} = \begin{pmatrix} 1 \\ 2 \\ 3 \end{pmatrix}\)     (b)   \(\mathbf{v} = \begin{pmatrix} 5 \\ -12 \\ 0 \end{pmatrix}\)     (c)   \(\mathbf{w} = \begin{pmatrix} 1 \\ 0 \\ 1 \end{pmatrix}\)

Solution

(a)   \(\hat{\mathbf{u}} = \dfrac{\mathbf{u}}{|\mathbf{u}|} = \dfrac{1}{\sqrt{14}} \begin{pmatrix} 1 \\ 2 \\ 3 \end{pmatrix} \begin{pmatrix} \frac{1}{\sqrt{14}} \\ \frac{2}{\sqrt{14}} \\ \frac{3}{\sqrt{14}} \end{pmatrix}\)

(b)   \(\hat{\mathbf{v}} = \dfrac{\mathbf{v}}{|\mathbf{v}|} = \dfrac{1}{13} \begin{pmatrix} 5 \\ -12 \\ 0 \end{pmatrix} = \begin{pmatrix} \frac{5}{13} \\ -\frac{12}{13} \\ 0 \end{pmatrix}\)

(c)   \(\hat{\mathbf{w}} = \dfrac{\mathbf{w}}{|\mathbf{w}|} = \dfrac{1}{\sqrt{2}} \begin{pmatrix} 1 \\ 0 \\ 1 \end{pmatrix} = \begin{pmatrix} \frac{1}{\sqrt{2}} \\ 0 \\ \frac{1}{\sqrt{2}} \end{pmatrix}\)

Python code#

The Sympy command for calculating the magnitude of a vector v is v.norm(). The code below calculates the solutions to example 1.3.

from sympy import *

# (i)
u = Matrix([1, 2, 3])
print("(i)")
display(u / u.norm())

# (ii)
v = Matrix([5, -12, 0])
print("(ii)")
display(v / v.norm())

# (iii)
w = Matrix([1, 0, 1])
print("(iii)")
display(w / w.norm())

(ii)
\[\begin{split}\displaystyle \left[\begin{matrix}\frac{5}{13}\\- \frac{12}{13}\\0\end{matrix}\right]\end{split}\]
(iii)
\[\begin{split}\displaystyle \left[\begin{matrix}\frac{\sqrt{2}}{2}\\0\\\frac{\sqrt{2}}{2}\end{matrix}\right]\end{split}\]
